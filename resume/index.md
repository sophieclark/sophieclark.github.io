## Projects

### Dissertation
For my dissertation I created a Recipe App for iOS written in Swift. The app has the functions to add, edit and delete recipes and also the ability to save them online so that they can be accessed from other devices. The app would also read out the recipe to the users to enable hands free usage and could set multiple timers for different parts of the recipe. The user could add their own photos for the different steps of the recipe. There was also a shopping list function which the user could add to manually or just simply tap an ingredient listed in a recipe to have it added to the users shopping list. The user could create an account by either entering their email and a password or by linking with Facebook.
